# Triage Labels

| Canonical role | Linear label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Requires evaluation |
| `needs-info` | `needs-info` | Waiting for information |
| `ready-for-agent` | `ready-for-agent` | Ready for AFK execution |
| `ready-for-human` | `ready-for-human` | Requires human intervention |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill mentions a canonical role, use the corresponding Linear label.
